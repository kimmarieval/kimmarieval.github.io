Component 06

FAQ Accordion Pattern

How would this work in a CMS?

Each section could map to a reusable snippet, content block, or template partial with editable fields for headings, copy, buttons, and images.

Why build components instead of one-off pages?

Components help improve consistency, reduce duplicate styling, and make it easier for marketing teams to launch pages quickly.

Can these sections be adapted for different campaigns?

Yes. The structure is intentionally flexible so the same layout patterns can support different offers, products, services, or lead-generation goals.
